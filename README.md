# 📱 Course Management App – React Native

A simple mobile application built as part of a university Mobile Application Development course.  
The app allows users to view and manage course information using React Native, navigation, and global state management.

---

## 🎯 Project Purpose

This project was developed to demonstrate core React Native concepts, including:

- Building multi-screen mobile applications
- Using React Navigation for screen transitions
- Managing global state with Context API
- Designing simple and responsive mobile UI components

---

## ✨ Features

- 📚 Display list of university courses  
- 📄 View detailed information for each course  
- 🧭 Navigation between Home and Course Details screens  
- 🌗 Light/Dark theme toggle  
- 🌍 Global state management using Context API  

---

## 🛠️ Tech Stack

- React Native  
- JavaScript  
- Context API  
- React Navigation

## 🚀 Getting Started

### Prerequisites
Make sure you have installed:
- Node.js
- npm or yarn
- Expo CLI (if using Expo) or React Native CLI

---

### Installation

Clone the repository:
```bash
git clone https://github.com/your-username/your-repo-name.git

Navigate into the project folder:
```bash
cd https://github.com/sewaroberry/React-Native-Course-Dashboard-App

Install dependencies:
```bash
npm install

Run the App

If using Expo:

expo start

If using React Native CLI:

npm run android
# or
npm run ios
